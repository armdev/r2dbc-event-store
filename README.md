# R2DBC EventStore [![R2DBC EventStore](https://github.com/daggerok/r2dbc-event-store/actions/workflows/ci.yaml/badge.svg)](https://github.com/daggerok/r2dbc-event-store/actions/workflows/ci.yaml)
